# Product Overview

**Sinara Soon** (AppKospin) is a mobile banking application for Kospin Sinara Artha, a cooperative financial institution. The app provides members with digital access to their financial accounts and services.

## Core Features

- Account management (savings, deposits, loans)
- Transaction history and balance inquiries
- Payment and billing services
- User authentication and profile management
- Multi-platform support (iOS, Android, Web)

## Target Platforms

- Android (primary)
- iOS
- Web (static output via Metro bundler)
